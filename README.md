# MealsToGo
A massive restaurant application. Along the way, I built a massive restaurant application using React, React Hooks, Expo, React Navigation, Firebase, Styled-Components, Google Maps/Places API and React Animations. This is a full iOS app.


## Features
1. Allowed users to search for more than 1000+ restaurants’ information including opening hours, locations and menus in their cities and provided users with the direction and location of restaurants by setting up Google Maps API. 
2. Built clean and beautiful UI with React-native and boosted website page rendering efficiency with React-Hooks and styled- components of React.  
3. Supported search function for user’s favorite restaurants and helped users save
4. Performed identification and authentication by applying the Firebase.
